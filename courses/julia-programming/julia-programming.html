Introduction to Julia Programing
Julia was originally released in 2012 by Alan Edelman, Stefan Karpinski, Jeff Bezanson, and Viral Shah. It is a free and open source programming language. Choosing a programming language is always subjective. For me, the following characteristics of Julia are decisive:

- Julia is developed as a high-performance programming language. Julia uses multiple dispatches, which allows the programmer to choose from different programming patterns adapted to the application. Julia is a dynamically typed language that can easily be used interactively. Julia has a nice high-level syntax that is easy to learn.

- Julia is an optionally typed programming language whose (user-defined) data types make the code clearer and more robust. Julia has an extended standard library and numerous third-party packages are available.

who should take this course
• This course is made to teach people who are experienced with other scripting languages the relatively new language Julia. Unlike the other course, this course assumes prior knowledge of some form of programming in a language such as Python, R, or MATLAB.
• We will start by introducing the basic syntax of the language, and quickly move into the details of how Julia is different from other scripting languages and how to exploit Julia's type system + multiple dispatches to be able to achieve C/Fortran-like performance while maintaining the concise syntax of a scripting language. Large parts of the package ecosystem will be explored and some information on implementation details will be highlighted in order for the participants to learn how to design Julia projects.
• The ideal participant is anyone who is interested in Julia.
• There can be many groups of people interested in using Julia. One large fraction can come with a strong software development background and C/Fortran knowledge and are looking to learn Julia as a tool to create packages with enhanced productivity while not losing performance. On the other side, there can be other users who are interested in the growing package ecosystem of Julia and would like to add Julia to their knowledge base. And then there's everything in between.
• One major goal of this course is to use Julia's language and syntax to bridge the gap between "package users" and "package developers" in the way that Julia has done.
