course content

- Introduction to the course, how the web works, differences between competitive programming (CP) and software development, how to build 'original' projects.
- Getting started with development - Setting up your system.
- Learning about Open-Source software (OSS) development, Open-Source vs Closed Source, Pros and Cons of OSS.
- Learning about Linux, why is it so crucial? Introduction to Linux commands.

- Web application and website system architecture - How does a website even work?
- Introduction to HTML 5 and writing your first website.
- How to make your website look beautiful? Cascading Style Sheets (CSS) fundamentals.
- Advanced HTML and CSS (including cross-browser support, `flexbox`, `grid`, `transform`, pseudo-classes etc)

- Understanding what is JavaScript and its ecosystem and how it works.
- Fundamentals of modern JavaScript including EcmaScript.
- Introduction to NodeJS and NPM. - Beginner JS: Variables, Object-Oriented JavaScript, Operators, Statements and Declaration
- Intermediate JS: `this` keyword, scope and closures, JS Objects, refactoring and code-splitting
- Advanced JS: Arrow Functions, Map/Filter/Reduce, Array and Object functions,
Destructuring, Spread Operator, and creating NPM modules.

- Introduction to Front-end Development with React
- Thinking the React Way - Basics, Fundamentals, and Comparisions.
- Building Components in React with Create-React-App
- Understanding React's architecture (including Virtual DOM, JSX, CSS-in-JS, NPM basics etc.) and writing our first React application.
- Passing `props` to components, Parent-Child relationship, Special Props (`children` and `rest`)
- Introduction to State Management and Hooks.
- Understanding Hooks in detail `useState`, `useEffect`, conditional rendering, forms, prop-drilling

- Understanding Node JS and Express, how servers work and communicate (HTTP methods)
- Setting up your own server with NodeJS, Express, and nodemon.
- Event Loops, Async/Await Patterns, and Promises.
- HTTP basics - GET, POST, PUT, DELETE (aka CRUD) operations explained
- What are APIs? Undestanding API structure.
- Routes, requests, responses, and JSON.
- Middleware in Express.
- Deploying server to the cloud using Heroku.

- What are databases and how do they store data
- Understanding Database paradigms like ACID (atomicity, consistency, isolation, durability) and Normality
- Introduction to NoSQL database MongoDB and MongoDB Atlas.
- Database Schema and Models using Mongoose and Node JS
- Setting up a MongoDB database and connecting it to our server.
- Mongoose and MongoDB CRUD operations using NodeJS and Express.

- Deploying the application we will build during this course to the cloud
- Setting up a repository for it on GitHub.
- Understanding the core parts of a good open-source project (including licencing, contribution guidelines, issue templates etc)
- Releasing our first production build.
